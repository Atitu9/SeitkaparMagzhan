<h5>Array
Array is a physical data structure with fixed size. It is used to store 3 predefined bank accounts.
LinkedList
LinkedList is used to store bank accounts because it is easy to add and remove elements dynamically.
Stack
Stack is used for transaction history because it follows the LIFO (Last In First Out) principle.
Queue
Queue is used for bill payments and account opening requests because it follows the FIFO (First In First Out) principle.
Brief Summary
In this assignment, I created a mini banking system in Java.
I used:
 LinkedList for storing bank accounts
 Stack for transaction history
 Queue for bill payments and account requests
 Array for fixed-size account storage
The program also contains: Bank Menu, ATM Menu,Admin Menu
During this assignment, I learned the difference between physical and logical data structures and how they are used in a real banking system.
